Deploy API Challenges
=====================

Build the deployable jar:

    mvn -B -pl challenger -am package

The deployable artifact is:

    challenger/target/apichallenges.jar

Run locally:

    java -jar challenger/target/apichallenges.jar -model=challengeapi -noshutdown -multiplayer

Docker:

    docker build -t eviltester/apichallenges -f docker/apichallenges/Dockerfile .
    docker run --rm -p 4567:4567 eviltester/apichallenges

Before deployment, run the ChallengerAuto suite against the target environment:

    mvn -B -pl challengerAuto -am test -Dchallenger.auto.target=existing -Dchallenger.auto.baseUrl=http://localhost:4567

For hosted environments, use:

    mvn -B -pl challengerAuto -am test -Dchallenger.auto.target=live -Dchallenger.auto.baseUrl=https://your-host.example

Only enable the full mutating suite against environments where creating,
updating, and deleting API data is acceptable:

    -Dchallenger.auto.external.full=true
